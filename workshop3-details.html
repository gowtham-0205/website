Welcome to the Web Development Basics Workshop

Welcome to the Web Development Basics Workshop

Embark on an exciting journey to explore the fundamental concepts of web development in this hands-on workshop. Whether you're an aspiring web developer or simply curious about creating dynamic and interactive websites, this workshop is designed to provide you with a solid foundation in modern web technologies.

About the Workshop

Discover the essentials of HTML, CSS, and JavaScript - the building blocks of the web. Learn how to craft responsive and user-friendly web interfaces that cater to a diverse audience across various devices.

Workshop Details

  • Date: September 10, 2023
  • Time: 10:00 AM - 1:00 PM (GMT)
  • Duration: 1 day
  • Location: Amrita Vishwa Vidyapeetham, Chennai Campus

Agenda

Get hands-on experience with:

  • HTML Structure and Elements
  • CSS Styling and Layout
  • Introduction to JavaScript
  • Creating Interactive Web Pages

About Session

Engage in practical sessions and learn from experienced web developers who will guide you through the process of creating functional and visually appealing websites.

Requirements

  • Basic computer literacy
  • No prior coding experience required

Project Submission

As part of the workshop, you'll have the opportunity to work on a practical web development project. The project submission deadline is September 30, 2023. This project will allow you to apply the skills you've learned and build a fully functional website using HTML, CSS, and JavaScript.

Certificate

Upon successful completion, you will receive a certificate of participation to showcase your newfound web development skills.
